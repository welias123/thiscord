Core Features

Built different.
By design.

Every feature exists because it should have been there from the start. No compromises, no surveillance.

⚡
Ultra-Low Latency
Rebuilt audio pipeline using native OS audio APIs. Voice packets are routed with sub-20ms end-to-end latency — eliminating the buffering overhead baked into stock Discord.
Audio Engine v2
🛡
Zero Telemetry
Every analytics call, fingerprinting hook, and usage beacon has been surgically removed. Your typing habits, active channels, and hardware profile stay on your machine — period.
Privacy First
🎨
Custom CSS Injection
A built-in theme engine with live-reload support. Write standard CSS, target any element. Full Stylus/SCSS transpilation pipeline included. Ship themes as single-file bundles.
Theme Engine
🧠
Memory Efficiency
60% lower RAM footprint vs. stock Electron builds. Achieved through renderer process consolidation, aggressive lazy-loading, and stripping 34 unused Chromium feature flags.
~120MB Idle
🔌
Plugin Support
A sandboxed plugin runtime with a stable public API. Install from the built-in registry or drop a .tcplugin file into your plugins folder. No CLI, no build step required.
Plugin SDK
🌐
Cross-Platform Native
Native builds for Windows 10/11 (x64, arm64) and major Linux distros (Ubuntu, Debian, Fedora). Each build is compiled with platform-specific optimizations — not just repackaged binaries.
Multi-Platform

Why ThisCord

Your client.
Your rules.

The official Discord client ships with tracking, forced updates, and a locked-down environment. ThisCord gives power users, developers, and privacy advocates the client they deserve.

🔒

Privacy by Architecture

Network calls are auditable. No hidden endpoints, no obfuscated JS. Every request is logged in the dev console so you know exactly what leaves your machine.

⚙

Developer-Grade Tooling

Full DevTools access in production, hot-reload for custom CSS, and a plugin SDK with TypeScript typings. Build your own features in an afternoon.

🚀

Performance Without Trade-offs

Native audio APIs, stripped Chromium flags, and a lean renderer process. Everything that doesn't serve you has been removed — not hidden behind a setting.

🌍

Community Driven

No corporate roadmap. Features are built by users, for users. Submit a PR, open an issue, or join the community — your voice shapes the next release.
